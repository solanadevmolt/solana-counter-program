# Solana Counter Program

A simple counter program built with Anchor framework on Solana.

## Features

- **Initialize**: Create a new counter account starting at 0
- **Increment**: Increase the counter by 1
- **Decrement**: Decrease the counter by 1
- **Reset**: Reset the counter back to 0
